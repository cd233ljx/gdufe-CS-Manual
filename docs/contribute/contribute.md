

#  **如何为本项目做出贡献**

> 你的每一次分享，都是照亮后来者的微光   
> 无论是修正错别字、补充食堂评分，还是分享课程笔记，都热烈欢迎！

---

##  **三种简单的贡献方式**

###  ✅方式一：GitHub 网页端编辑
1. 在网站任意页面点击右上角 `Edit this page`（需登录 GitHub）  
2. 直接修改 Markdown 内容（支持实时预览）  
3. 滚动到底部 → 填写修改说明 → 点击 **Propose changes**  
4. 提交 Pull Request（PR），我们会快速审核合并！  
 *适合：修正错别字、补充小贴士、更新链接*

###  ✅方式二：提交 Issue
1. 访问 [Issues 页面](https://github.com/cd233ljx/gdufe-CS-Manual/issues)  
2. 点击 **New Issue** → 选择模板：  
   -  `bug-report`：发现错误/失效链接  
   -  `feature-request`：建议新增内容（如“求补充XX课程资源”）  
   -  `food-suggestion`：美食投稿（附文字+图片）  
3. 填写清晰描述 → 提交！  


###  ✅方式三：邮件投稿
📧 发送内容至：**utopiacd@student.gdufe.edu.cn**  
标题格式：`[投稿] 类型+主题`（例：`[投稿] 美食-三水校外糖水店`）  
正文请包含：  
```markdown
- 投稿类型：课程资源 / 美食推荐 / 生活贴士 / 其他
- 具体内容：（文字+图片链接/描述/附件）
- 你的称呼：（可选，用于致谢）
- 联系方式：（可选，方便我们确认细节）
```


##  **贡献内容规范（可选）**

### 📚 课程资源投稿
```markdown
## 课程名称
- **建议学期**：大二上  
- **资源内容**：  
  ▸ 课件/实验指导（附网盘链接+提取码）  
  ▸ 期末整理 
  ▸ 学习心得（老师习惯/避坑指南）  
- **投稿人**：2025级张三（可选）  
```
> ⚠️ 注意：  
> - 避免直接上传大文件（用网盘链接，建议使用**夸克/阿里网盘**）  
> - 尊重版权：勿分享受版权保护的完整教材  
> - 标注资源来源

### 🍜 美食投稿模板
```markdown
### [店名]（例：湘菜馆木桶饭）
 位置：三水校区西门荣创产业园 
 人均：15元  
 推荐：农家一碗香 
 评价：神 
 图片：[上传至图床后粘贴链接]  
```
