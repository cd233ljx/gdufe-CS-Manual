# 本科培养方案总则

一、制定依据

本本科培养方案依据《中华人民共和国高等教育法》《普通高等学校本科专业类教学质量国家标准》，结合国家和粤港澳大湾区经济社会发展对计算机领域高级工程应用型人才的需求，立足广东财经大学办学定位和大数据与人工智能学院发展规划，遵循计算机科学与技术学科发展规律，结合专业特色制定，适用于大数据与人工智能学院计算机科学与技术专业全日制普通本科学生。

二、培养理念

坚持“德、智、体、美、劳全面发展”的教育方针，以立德树人为根本任务，突出“厚基础、强实践、重创新、促融合”的培养理念，注重培养学生的计算思维、专业能力和综合素质，兼顾理论教学与实践应用，着力打造适应IT行业未来发展需要，能够服务国家和区域经济发展的高级工程应用型专业人才。

三、适用范围

本方案适用于大数据与人工智能学院计算机科学与技术专业全日制普通本科在校生（含广州校区、佛山校区），涵盖学生从入学到毕业的全过程培养，包括课程修读、实践教学、学分要求、毕业条件、学位授予等所有相关环节。港澳台学生除本方案明确的特殊规定外，参照本方案执行。

四、培养目标

本专业培养适应国家和粤港澳大湾区经济发展需求，德、智、体、美、劳全面发展，掌握数学与自然科学基础知识及计算机科学相关的基本理论、基本知识和基本方法，具有较强的专业能力和良好的综合素质，能够胜任政府机关、企事业单位、科研院所中计算机系统分析、设计、开发及研究等相关工作，并能适应IT未来发展需要的高级工程应用型专业人才。

五、培养要求

（一）知识要求

1. 了解与本专业相关的职业和行业的重要法律法规及方针政策，理解工程技术与信息技术应用相关的伦理的基本要求；

2. 掌握计算学科的基本思维方法和研究方法，具有良好的科学素养和一定的工程意识，掌握从事本专业工作所需的数学和其他相关的自然科学知识；

3. 系统掌握计算机科学与技术学科的基础理论和专业知识，理解本学科的基本概念、知识结构和典型方法，了解计算机科学与技术学科的发展现状和趋势。

（二）能力要求

1. 培养计算思维能力，能运用计算机科学的概念和方法去求解问题、设计系统，能够综合运用所掌握的知识、方法和技术解决实际问题；

2. 培养专业实践能力，可以设计和实现软件系统、硬件系统及各类计算机应用系统；

3. 具备终身学习意识以及运用现代技术获取相关信息和新技术、新知识的能力，具有创新意识，并具有技术创新的初步能力，具有一定的组织管理能力、独立工作能力、人际交往能力和团队合作能力。

（三）素质要求

1. 具有良好的思想品德和较高的道德情操，热爱社会主义祖国，拥护中国共产党的领导，掌握马列主义、毛泽东思想与中国特色社会主义基本理论；

2. 身体健康，具有良好的人文社会科学素养、职业道德和心理素质，社会责任感强；

3. 能独立思考，坚持理论联系实际、实事求是和谦虚谨慎、认真细致的科学态度，自觉自愿地遵守职业道德规范。

六、修业年限与学位授予

1. 标准修业年限：四年，实行弹性修业年限，学生可在3-6年内完成学业；

2. 学位授予：符合毕业条件和学位授予条件者，授予工学学士学位。

七、方案实施与调整

本方案由大数据与人工智能学院负责组织实施，严格按照教学计划开展教学、实践等各项工作。若因国家政策调整、行业发展需求变化或学科建设需要，需对本方案进行修订，由学院组织论证后，报学校教务处审批备案后执行。

八、附则

本方案自发布之日起生效，由大数据与人工智能学院负责解释。辅修本专业的学生，参照本方案中辅修相关规定执行。
