# 通识课要求

一、课程定位

通识课程是计算机科学与技术专业本科培养的重要组成部分，旨在拓宽学生知识面、提升人文素养、培养科学精神和社会责任感，兼顾思想引领、基础素养和综合能力提升，为专业学习和终身发展奠定坚实基础，分为通识必修课和通识选修课两大类。

二、通识必修课要求（51学分）

（一）课程模块与学分分配

1.  思政一类（14.5学分）：马克思主义基本原理（3学分）、军事理论（2学分）、形势与政策I-VI（合计3学分）、思想道德与法治（2学分）、习近平新时代中国特色社会主义思想概论（3学分）、毛泽东思想和中国特色社会主义理论体系概论（2学分）、中国近现代史纲要（3学分）；

2.  思政二类（2学分）：从党史、新中国史、改革开放史、社会主义发展史四门课程中选择一门修读，每门课程2学分；

3.  综合类（8.5学分）：国家安全教育（1学分）、大学生心理健康教育（2学分）、职业生涯与发展规划（0.5学分）、大学语文（2学分）、大学生劳动教育（2学分）、体质健康教育（1学分）、湾区财经概论（2学分）、就业指导（0.5学分）、创业基础（2学分）；

4.  外语类（10学分）：根据高考外语语种分为英语和日语两类，具体修读要求如下：

（1）高考为英语的学生：实行高阶班、标准班分层教学，高阶班修读大学英语III、大学英语IV、英汉/汉英翻译、英语高阶读写；标准班修读大学英语II、大学英语III、财经英语、职场英语，均为10学分；

（2）高考为日语的学生：普通专业修读大学日语I、大学日语II、大学日语III、大学日语IV，合计10学分；

5.  体育类（8学分）：具体要求详见《体育课要求.md》。

（二）修读规则

1.  所有通识必修课均为必修课程，必须修读并考核合格，方可获得对应学分，未合格者需重修；

2.  课程修读学期参照教学进度计划，建议按学期修读，避免集中修读导致学分压力过大；

3.  思政二类课程需在1-8学期内完成一门修读，可根据自身课程安排灵活选择修读学期；

4.  外语类课程实行分层分类教学，学生入学后依据高考成绩遴选进入对应课程体系修读，不得擅自更改课程体系；

5.  体质健康教育课程在3-4学期开设，考查合格方可获得学分，与体质健康测试相关联。

三、通识选修课要求（10学分）

（一）学分与模块要求

1.  总学分要求：至少修读10学分，其中文学与艺术模块至少修读2学分，未达到模块要求者，视为通识选修课学分不合格；

2.  课程模块：共设置十大模块，分别为思想与政治、运动与健康、自然与科技、文学与艺术、创新与创业、表达与沟通、思维与方法、法治与社会、湾区视野、跨学科与跨专业模块。

（二）课程选择规则

1.  思想与政治、运动与健康、自然与科技、文学与艺术、创新与创业、表达与沟通、思维与方法、法治与社会、湾区视野九大模块，课程设置见《广东财经大学2024年版通识选修课程设置一览表》；

2.  跨学科、跨专业模块课程，见其他专业人才培养方案中开设的学科基础课和专业课（不含同一专业不同方向的课程）；

3.  学生可根据自身兴趣、职业规划和知识需求，自主选择不同模块的课程，修读学分可累计计算；

4.  通识选修课可在1-6学期内完成修读，建议合理分配各学期修读学分，避免集中在最后学期修读。

四、港澳台生特殊要求

1.  港澳台学生须修读“中国概况”（4学分，第5学期开设）、“中国特色社会主义理论与实践”（3学分，第6学期开设）两门国情类必修课，可免修通识必修课中的思政一类课程；

2.  港澳台学生可在国情教育学习平台学习相应课程作为国情类选修课，获得学时证明后，可减免通识选修课程总学分2学分，具体说明见教务处网站通知。

五、考核与学分认定

1.  通识必修课考核方式分为考试和考查两种，具体考核方式参照教学进度计划，考核合格方可获得学分；

2.  通识选修课考核方式由任课教师确定，考核合格即可获得对应学分，不合格者可重修或更换其他课程修读；

3.  通识课程学分不计入专业核心学分，但属于毕业必备学分，未修满规定学分者，无法毕业。
