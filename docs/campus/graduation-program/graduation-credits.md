# 毕业学分毕业条件说明

一、毕业学分要求

（一）主修学生（全日制普通本科）

1.  总学分要求：需修满160学分（不含第二课堂学分），各模块学分分配必须符合以下要求，缺一不可：

（1）通识必修课：51学分（含思政类、综合类、外语类、体育类课程）；

（2）通识选修课：10学分，其中文学与艺术模块≥2学分；

（3）学科基础课：40学分（全部为必修课程）；

（4）专业必修课：51学分；

（5）专业选修课：8学分。

2.  第二课堂学分：除港澳台学生外，每个学生还须取得10个第二课堂学分，具体要求见《广东财经大学“第二课堂成绩单”制度实施办法（试行）》（粤财大〔2019〕42号），第二课堂学分不纳入160学分统计，但属于毕业必备条件。

（二）辅修学生

需修满45学分，其中学科基础课30学分、专业必修课15学分，具体课程参照《教学计划进度》中适用于辅修专业的所有课程，完成辅修课程考核合格后，可获得学校辅修专业证书，不影响主修专业毕业。

二、毕业基本条件

学生需同时满足以下所有条件，方可准予毕业：

1.  学分达标：主修学生修满160学分（不含第二课堂），第二课堂学分（非港澳台生）达标；辅修学生修满45学分（辅修专用）；

2.  考核合格：所有修读课程（含必修、选修）考核合格，无不合格课程未重修通过情况；毕业实习、毕业设计（论文）考核合格；

3.  体质达标：按教育部《国家学生体质健康标准》完成每年体质健康测试，且测试达标，补测仍不达标者，不予毕业；

4.  思想品德合格：思想品德考核合格，无严重违纪处分，恪守学术道德，无学术不端行为（如抄袭、作弊等）；

5.  港澳台生额外要求：完成“中国概况”“中国特色社会主义理论与实践”两门国情类必修课修读，考核合格，免修思政一类课程的需按规定完成国情类课程修读。

三、学分认定与补修规则

1.  课程学分认定：所有课程需考核合格方可获得对应学分，不合格课程需按学校规定参加重修，重修考核合格后，方可认定学分；

2.  补修要求：若某一模块学分未达到规定要求，需补修对应模块的课程，直至达到学分要求；通识选修课未达到模块要求（文学与艺术≥2学分）的，需补修该模块课程；

3.  缓考与重修：因病假、公假等特殊情况无法参加考试的，可按规定申请缓考；无故缺考、考试作弊者，按旷考、作弊处理，该课程成绩记为不合格，需重修；

4.  学分替换：原则上不允许学分替换，特殊情况（如课程停开）需按学校教务处规定办理学分替换手续，替换课程的学分、难度需与原课程相当，经学院和教务处审批同意后方可认定。

四、毕业状态说明

1.  正常毕业：满足所有毕业条件，修满规定学分，考核合格，体质达标，思想品德合格，准予正常毕业，发放毕业证书；

2.  延期毕业：因学分未修满、课程不合格、体质测试不达标等原因，未满足毕业条件的，可申请延期毕业，延期期限按学校相关规定执行，延期期间需补修相关课程、完成体质测试，直至满足毕业条件；

3.  结业：修业年限届满，仍未修满规定学分或未达到毕业条件，但已修学分达到总学分的90%以上，可发放结业证书，结业后可在规定期限内回校补修，补修合格后，可换发毕业证书；

4.  肄业：修业年限届满，未修满总学分的90%，或因严重违纪、学术不端等原因被取消毕业资格的，发放肄业证书，不予换发毕业证书；

5.  退学：因长期缺勤、学分未达标、严重违纪等原因，按学校学籍管理规定被退学的，不予发放毕业证书，按退学相关规定处理。

五、注意事项

1.  学生应定期查询自身学分修读情况，对照培养方案，合理安排课程修读计划，避免出现学分不足、模块不达标等问题；

2.  毕业实习、毕业设计是毕业必备环节，未完成或考核不合格者，无法毕业，学生需按教学计划按时完成；

3.  第二课堂学分需在毕业前完成认定，未达到10学分的，需补充相关实践活动，直至达标；

4.  港澳台学生需严格遵守国情类课程修读要求，未完成相关课程修读或考核不合格者，无法毕业。
