# 演示账号所属专业年级的培养方案

一、基本信息

1.  演示账号信息：姓名（演示）、学号（演示）、年级：2025级、专业：计算机科学与技术、学院：大数据与人工智能学院、校区：广州校区（可根据实际调整为佛山校区）；

2.  专业归属：计算机类；

3.  修业年限：四年（2025年9月-2029年6月）；

4.  学位授予：工学学士；

5.  核心培养定位：以粤港澳大湾区IT产业发展需求为导向，培养具备扎实的计算机科学理论基础和较强的实践开发能力，能够从事计算机系统分析、设计、开发及相关研究工作，适应IT行业快速发展的高级工程应用型人才。

二、培养目标与要求

（一）培养目标

与本科培养方案总则一致，聚焦2025级学生成长周期，结合未来4年IT行业发展趋势，强化实践能力和创新意识培养，确保毕业时具备符合行业需求的专业素养和职业能力，能够快速适应岗位需求，同时具备终身学习和技术提升的潜力。

（二）培养要求

严格遵循本专业培养方案的知识、能力、素质要求，结合2025级学生特点，重点强化以下方面：

1.  知识层面：扎实掌握高等数学、线性代数等自然科学基础知识，系统学习计算机科学导论、数据结构、操作系统等核心课程，了解人工智能、大数据等前沿技术；

2.  能力层面：重点培养编程开发、系统设计、问题解决能力，通过课程设计、实习实训、学科竞赛等提升实践能力和创新能力，培养团队协作和沟通表达能力；

3.  素质层面：强化思想品德教育和职业道德培养，注重人文素养和社会责任感提升，培养严谨的科学态度和求真务实的工作作风。

三、核心课程与修读安排

（一）核心课程

计算机科学导论、程序设计基础、数据结构、计算机组成原理、操作系统、计算机网络、编译原理、数据库原理、算法分析与设计、软件工程、离散数学。

（二）分学期修读规划（参考）

1.  第1学期（2025年9月-2026年1月）：主修通识必修课（马克思主义基本原理、军事理论等）、学科基础课（高等数学I、程序设计基础）、专业必修课（计算机科学导论），合计26.0学分；

2.  第2学期（2026年2月-2026年6月）：主修通识必修课（思想道德与法治、大学语文等）、学科基础课（高等数学II、大学物理）、专业必修课（面向对象程序设计等），合计26.5学分；

3.  第3学期（2026年9月-2027年1月）：主修通识必修课（习近平新时代中国特色社会主义思想概论等）、学科基础课（线性代数a、数据结构等）、专业必修课（离散数学等）、通识选修课，合计28.5学分；

4.  第4学期（2027年2月-2027年6月）：主修通识必修课（中国近现代史纲要等）、学科基础课（概率论与数理统计b、操作系统等）、专业必修课（编译原理等）、通识选修课，合计25.5学分；

5.  第5学期（2027年9月-2028年1月）：主修通识必修课（形势与政策V等）、学科基础课（计算机网络）、专业必修课（算法分析与设计、软件工程等）、专业选修课、通识选修课，合计23.0学分；

6.  第6学期（2028年2月-2028年6月）：主修通识必修课（形势与政策VI等）、专业必修课（AI处理器架构与编程等）、专业选修课、通识选修课，合计19.5学分；

7.  第7学期（2028年9月-2029年1月）：专业选修课、毕业实习，合计6.0学分；

8.  第8学期（2029年2月-2029年6月）：毕业设计，合计5.0学分。

四、实践教学安排

1.  课程实践：各核心课程配套实验、实训环节，如程序设计基础实验、数据结构实验、操作系统实验等，培养基础实践能力；

2.  课程设计：第3-6学期依次完成编程拓展课程设计、数据结构课程设计、综合课程设计I、综合课程设计II、智能系统课程设计等，强化系统设计和开发能力；

3.  社会实践：第4学期开展湾区经济与社会实践调查I，第6学期开展湾区经济与社会实践调查II，提升社会适应能力和调研能力；

4.  毕业实习：第7学期开展为期10周的毕业实习，深入企业一线，将理论知识与实际工作结合，积累岗位经验；

5.  毕业设计：第8学期完成毕业设计（5学分），培养综合运用专业知识解决实际问题的能力，完成学业总结。

五、学分要求

1.  主修学分：需修满160学分（不含第二课堂学分），其中通识必修课51学分、通识选修课10学分（文学与艺术模块≥2学分）、学科基础课40学分、专业必修课51学分、专业选修课8学分；

2.  第二课堂学分：需取得10个第二课堂学分，具体要求参照《广东财经大学“第二课堂成绩单”制度实施办法（试行）》（粤财大〔2019〕42号）；

3.  体质健康要求：每年参加体质健康测试，测试达标方可毕业。

六、毕业与学位相关说明

1.  毕业条件：修满规定学分，所有课程考核合格，体质健康测试达标，思想品德合格，完成毕业实习和毕业设计并考核合格；

2.  学位授予：符合毕业条件，且达到学位授予相关要求，授予工学学士学位；

3.  特殊说明：若期间出现课程不合格，需按学校规定参加重修，直至考核合格；若需请假，严格按照学校请假制度执行，避免影响修读进度。
