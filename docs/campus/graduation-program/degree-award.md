# 学位授予相关基本规则

一、学位授予依据

依据《中华人民共和国学位条例》《中华人民共和国学位条例暂行实施办法》《广东财经大学学士学位授予工作实施办法》，结合大数据与人工智能学院计算机科学与技术专业培养方案，制定本规则，适用于本专业全日制普通本科毕业生。

二、授予学位类型

本专业授予工学学士学位，符合学位授予条件的毕业生，由广东财经大学授予工学学士学位，颁发学位证书。

三、学位授予基本条件

学生需同时满足以下所有条件，方可授予工学学士学位：

1.  符合毕业条件：已修满本专业规定的160学分（不含第二课堂），第二课堂学分（非港澳台生）达标，所有课程考核合格，毕业实习、毕业设计（论文）考核合格，体质健康测试达标，思想品德合格，无严重违纪处分和学术不端行为；

2.  专业能力达标：掌握本专业核心理论、基本知识和基本技能，具备一定的实践能力和创新意识，能够运用专业知识解决简单实际问题，毕业设计（论文）达到相应水平；

3.  外语水平要求：达到学校规定的学士学位授予外语水平要求（具体按学校当年发布的外语水平认定标准执行）；

4.  无影响学位授予的违纪违规行为：无严重违纪处分（如记过及以上处分），无学术不端行为（如毕业设计抄袭、论文作弊、课程考试作弊等），若有上述行为，取消学位授予资格；

5.  港澳台学生额外要求：完成“中国概况”“中国特色社会主义理论与实践”两门国情类必修课修读，考核合格，方可授予学位。

四、学位授予程序

1.  毕业生申请：学生在毕业前，对照学位授予条件，向所在学院（大数据与人工智能学院）提交学士学位授予申请；

2.  学院审核：学院学位评定分委员会对申请人的学分、考核成绩、思想品德、外语水平、毕业设计（论文）等情况进行全面审核，提出拟授予学位、暂缓授予学位或不授予学位的意见；

3.  学校审批：学校学位评定委员会对各学院提交的拟授予学位名单进行审核，审批通过后，正式授予学士学位；

4.  证书发放：学校统一印制学位证书，由学院发放给符合条件的毕业生，未授予学位者，不发放学位证书。

五、暂缓授予学位、不授予学位的情形

（一）暂缓授予学位的情形

1.  外语水平未达到学校规定要求，但符合毕业条件的，可暂缓授予学位，给予一定期限（一般为1-2年）补测外语，补测合格后，可申请授予学位；

2.  毕业设计（论文）达到合格标准，但未达到学位授予相关要求的，可暂缓授予学位，允许修改毕业设计（论文）后重新审核，审核合格后，可申请授予学位；

3.  有轻微违纪行为，未达到取消学位授予资格标准，但需整改的，暂缓授予学位，整改合格后，可申请授予学位。

（二）不授予学位的情形

1.  未达到毕业条件，无法正常毕业的；

2.  有严重违纪处分（记过及以上），或有学术不端行为（如抄袭、作弊等），经学校学位评定委员会审核，取消学位授予资格的；

3.  外语水平经补测后仍未达到学校规定要求的；

4.  毕业设计（论文）修改后仍未达到学位授予相关要求的；

5.  港澳台学生未完成国情类必修课修读或考核不合格的；

6.  其他不符合学位授予条件，经学校学位评定委员会审核确定不授予学位的情形。

六、学位授予相关补充说明

1.  学位授予与毕业是两个独立环节，符合毕业条件不一定能获得学位，需同时满足学位授予条件；

2.  暂缓授予学位的期限，按学校相关规定执行，逾期未达到学位授予条件的，视为自动放弃学位申请，不再授予学位；

3.  若学生对学位授予结果有异议，可按学校相关规定，向学院和学校学位评定委员会提出申诉，申诉流程按学校申诉管理办法执行；

4.  学位证书一经发放，不得涂改、伪造，若有遗失，可按学校规定申请补办学位证明书，学位证明书与原学位证书具有同等效力；

5.  本规则未尽事宜，按《广东财经大学学士学位授予工作实施办法》执行，若学校调整学位授予相关规定，以最新规定为准。
