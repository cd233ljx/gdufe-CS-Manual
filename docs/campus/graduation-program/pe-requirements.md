# 体育课要求

一、课程定位与学分要求

体育类课程为通识必修课，是本科培养中“体”育教育的核心组成部分，旨在增强学生体质、培养体育兴趣、养成终身锻炼习惯，提升身体素质和心理素质，为专业学习和终身发展提供保障。

总学分要求：8学分，均为体育选项课学分，不计入第二课堂学分，是毕业必备学分，未修满8学分或体质健康测试不达标者，无法毕业。

二、修读安排

（一）修读学期

建议修读学期为第1-2学期和第5-6学期，每学期修读1门体育选项课，逐步完成8学分修读，学生可根据自身课程安排适当调整，但需在毕业前完成所有学分修读。

（二）课程设置

具体课程设置详见《广东财经大学体育必修选项课程设置一览表》，涵盖各类体育项目（如球类、田径、武术、健身等），学生可根据自身兴趣、身体素质自主选择课程。

三、选课规则

1.  每学期限选1门体育选项课，选课采用“先到先得”原则，按学校选课系统规定的时间和流程操作，逾期未选课者，视为自动放弃该学期体育课程修读，需后续学期补修；

2.  同一体育选项课程，若考试通过，不得再次重修或选修，避免重复修读；

3.  身体不适、无法正常参加体育选项课的学生，可凭三甲医院出具的相关证明，向学院和体育部申请修读保健课，修读要求、考核标准按学校相关规定执行；

4.  体育课程选课成功后，不得擅自退课（特殊情况除外），退课需按学校选课退课流程办理，无故退课者，视为该课程考核不合格。

四、考核与成绩认定

（一）体育选项课考核

每门体育选项课考核分为平时考勤、课堂练习、技能测试三个环节，具体考核权重由任课教师确定，考核合格方可获得对应学分（每门课程一般为2学分），不合格者需重修。

（二）体质健康测试

1.  根据教育部《国家学生体质健康标准》，学生每年需参加一次体质健康测试，测试时间由学校体育部统一安排；

2.  体质健康测试达标是毕业的必备条件之一，未达标者需按学校要求参加补测，直至达标，补测仍不达标者，无法毕业；

3.  体质健康教育课程（1学分/学期，计入通识必修课）在第3-4学期开设，考查合格方可获得学分，该课程与体质健康测试相关联，辅助学生提升体质，掌握科学锻炼方法。

五、注意事项

1.  体育课程需按时上课，无故缺勤、迟到、早退者，按学校考勤规定处理，缺勤次数过多将影响课程考核成绩；

2.  参加体育课程和体质健康测试时，需做好热身准备，避免运动损伤，身体出现不适时，应及时向任课教师或测试工作人员报告；

3.  体育学分修读需循序渐进，建议按学期合理安排，避免集中在毕业前突击修读，影响修读效果和成绩；

4.  若因特殊情况（如伤病、休学）无法按时参加体育课程修读或体质健康测试，需提前向学院和体育部报备，按学校相关规定办理缓修、缓测手续。
