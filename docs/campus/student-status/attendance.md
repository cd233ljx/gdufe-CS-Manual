# 考勤相关说明

## 一、官方依据（仅限近三年教务处/学工部官方，无二级学院、无老旧文件）

1. 本科生日常考勤与课堂管理相关要求 | 2024-09 广东财经大学学工部

2. 普教本科生课程学习与课堂考勤管理规定 | 2025-03 广东财经大学教务处

3. 学生违纪处分管理补充通知 | 2025-10 学校职能部门官方发布

## 二、适用范围

广东财经大学全日制普通本科在校生，广州、佛山两校区统一执行；适用于日常上课、实验课、实践实训、集体教学活动、军训、讲座等全部教学环节。

## 三、基础考勤总则

1. 所有教学活动实行**全员考勤**，由任课教师、班委、学院统一登记记录。

2. 考勤结果纳入课程平时成绩、综合素质测评、学籍管理考核。

3. 未提前办理合法请假手续，无故缺席、迟到、早退，一律按违规考勤处理。

4. 公假、病假、事假经审批通过，不计缺勤、不计旷课。

## 四、迟到、早退、缺勤界定

1. **迟到**：上课开始后15分钟内进入课堂。

2. **早退**：课程结束前15分钟内擅自离开课堂。

3. **严重迟到/缺课**：上课迟到超15分钟，直接按**旷课1学时**认定。

4. **擅自离校**：课堂中途未经允许私自离开，按旷课处理。

5. 多次迟到、早退可累计折算为旷课学时。

## 五、旷课认定标准

1. 无请假手续、请假未获批、假期超期未销假，擅自缺课即为旷课。

2. 理论课：按实际授课学时统计旷课。

3. 实验课、实训课、体育课：单次缺席直接按整节学时记旷课。

4. 学校统一组织必修活动（入学教育、学风教育、集体教学）无故缺席，等同旷课。

## 六、请假与考勤关联规则

1. 所有缺勤必须**事前完成线上/线下审批**，紧急情况48小时内补手续。

2. 审批通过的病假、事假、公假，考勤记录正常，不记缺勤、不扣分。

3. 缓考、因病休养等合理报备，不纳入日常旷课统计。

4. 事后补假无合理证明、材料造假，一律驳回，仍按旷课处理。

## 七、考勤与课程成绩挂钩

1. 课堂出勤占课程平时分固定比例，无故缺勤直接扣减平时成绩。

2. 单门课程**旷课、缺勤累计超标**，任课教师可取消该生课程考核资格，成绩记为不合格，需重修。

3. 迟到、早退累计次数达标，同步扣除平时表现分。

## 八、旷课违纪处分标准

以**一学期累计旷课学时**为判定依据：

1. 10–29学时：警告

2. 30–39学时：严重警告

3. 40–49学时：记过

4. 50–59学时：留校察看

5. 60学时及以上：按退学相关规定处理

## 九、特殊场景考勤要求

1. **实验/实践课**：管理更严格，不得无故缺岗、中途离岗，缺勤直接影响实践成绩。

2. **体育课**：无合法免修、保健审批，无故缺课按旷课处理，影响体育必修学分。

3. **军训、集中实训**：全程封闭考勤，无特殊审批不得请假，违规记入档案。

4. **外出交流/公假**：凭学校盖章公假证明，全程免考勤处罚，事后自行补修遗漏课程。

## 十、考勤申诉与整改

1. 考勤记录有误，可凭上课凭证、行程记录向任课教师+学院教学秘书申请核对更正。

2. 超期旷课记录原则上不可撤销，仅合法审批假条可修正考勤。

3. 长期因病、特殊身体情况，可凭医院证明申请长期考勤备案。

## 十一、通用注意事项

1. 严禁代签到、代打卡、虚假签到，一经查实，双方均按违纪+旷课双重处理。

2. 每学期末学院统一汇总考勤数据，纳入学籍预警。

3. 考勤异常预警学生，需限期整改，否则影响评奖评优、入党、奖助申请。
