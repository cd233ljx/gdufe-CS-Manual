## 二、适用范围

广东财经大学全日制普通本科在校生（广州校区与佛山校区统一执行），**不含研究生、继续教育学生**。适用于所有需要请假的场景，包括：

1. 日常教学活动（课程、实验、实习等）

2. 学校统一组织的集体活动（军训、运动会、讲座等）

3. 报到注册、考试等重要教学节点

4. 其他经学校批准的请假事项

## 三、请假类型与申请条件（核心速查）

| 请假类型 | 申请条件 | 必备材料 | 特殊说明 |

|----------|----------|----------|----------|

| 病假 | 因身体不适无法参加教学活动 | 校医院或二级甲等及以上医院诊断证明、病历 | 急病可先口头请假，返校后48小时内补材料 |

| 事假 | 因重要个人事务必须处理（一般从严掌握） | 详细事由说明、必要证明材料、家长知情同意书 | 一般不批准旅游、娱乐等非必要事由 |

| 公假 | 因学校/学院安排参加比赛、会议、实习等 | 学校/学院出具的公假证明 | 公假期间不影响考勤与成绩 |

| 婚假 | 达到法定结婚年龄并办理结婚登记 | 结婚证复印件 | 按国家规定给予3天假期 |

| 丧假 | 直系亲属（父母、配偶、子女、祖父母、外祖父母）去世 | 亲属关系证明、死亡证明 | 给予3天假期，路程远可适当延长 |

| 休学相关请假 | 因病需请假治疗超过6周或一学期累计请假超6周 | 医院诊断证明、休学申请表 | 按休学流程办理，详见休学/复学规则 |

## 四、请假审批权限与期限（核心速查）

| 请假时长 | 审批权限 | 备注 |

|----------|----------|------|

| 3天以内（含3天） | 辅导员（班主任）审批 | 线上通过企业微信"学工通"或融合门户办理 |

| 4-7天（含7天） | 学院主管教学领导审批 | 需提交书面申请及相关证明材料 |

| 8天以上 | 学院主管教学领导签署意见后，报教务处审批 | 一学期累计请假最长不超过6周（含6周） |

| 报到注册请假 | 学院审批后报教务处备案 | 假期一般不超过两周，逾期未报到按退学处理 |

| 考试期间请假 | 教务处审批 | 一般不批准，特殊情况需提供充分证明 |

## 五、请假流程（线上+线下）

### （一）线上请假流程（推荐）

1. 登录企业微信→"学工通"→"请假申请"

2. 填写请假类型、事由、起止时间、请假天数

3. 上传相关证明材料（如病假需上传医院证明）

4. 提交申请，等待审批（按权限逐级审批）

5. 审批通过后，系统自动通知辅导员、任课教师

6. 请假期间如需延假，在系统中提交延假申请

### （二）线下请假流程（适用于特殊情况）

1. 下载并填写《广东财经大学学生请假审批表》

2. 附相关证明材料（病假需校医院或指定医院证明）

3. 按审批权限逐级签字：
- 3天内：辅导员（班主任）签字
- 4-7天：辅导员+学院主管教学领导签字
- 8天以上：辅导员+学院主管教学领导+教务处签字

4. 交学院办公室备案，同时通知任课教师

5. 请假期满后，到学院办公室销假

### （三）特殊情况处理

1. **急病/紧急事故**：可先通过电话、微信等方式向辅导员口头请假，返校后48小时内补全书面材料和审批手续

2. **异地请假**：无法当面办理时，可通过线上流程申请，或委托同学代办（需提供委托书）

3. **集体活动请假**：参加军训、运动会等集体活动，需额外向活动组织部门报备

## 六、销假与延假规则

### （一）销假流程

1. 请假期满后，学生应及时返校

2. 线上请假：在企业微信"学工通"中提交销假申请

3. 线下请假：到学院办公室办理销假手续

4. 未按时销假且未申请延假的，超期部分按旷课处理

### （二）延假规则

1. 需延长假期的，应在原假期结束前提出申请

2. 延假申请流程与请假流程相同，需重新提交证明材料

3. 延假时长计入累计请假时长，不得超过一学期6周上限

---

## 七、考勤与旷课处理规定

### （一）考勤计算标准

1. 行课日旷课一天按实际授课时数计

2. 实践性教学活动（实验、实习、军训等）旷课半天按2学时计，一天按4学时计

3. 学校规定必须参加的集体活动（如开学典礼、毕业典礼），无故缺席按实际时长计旷课

### （二）旷课处分标准

| 一学期旷课累计学时 | 处分类型 |

|--------------------|----------|

| 10-29学时 | 警告处分 |

| 30-39学时 | 严重警告处分 |

| 40-49学时 | 记过处分 |

| 50-59学时 | 留校察看处分 |

| 60学时及以上 | 退学处理 |

### （三）请假与成绩影响

1. 请假期间错过的课程，学生应主动联系任课教师补课或自学

2. 因病假缺考，可申请缓考，按缓考规定处理

3. 因事假缺考，一般不允许缓考，按旷考处理

4. 公假期间缺考，可凭公假证明申请正常补考

---

## 八、通用规则与注意事项

1. **事先申请原则**：除急病或紧急事故外，均须事先办理请假手续，严禁事后补假（特殊情况除外）

2. **材料真实性**：所有证明材料必须真实有效，**弄虚作假者取消请假资格**，并按《广东财经大学学生违纪处分规定》处理

3. **通讯畅通**：请假期间应保持通讯畅通，以便接收学校通知

4. **咨询方式**：
- 教务处咨询电话：广州校区020-84096837，佛山校区0757-87200316
- 工作时间：周一至周五 9:00-12:00，14:30-17:00
- 线上咨询：企业微信"学工通"或融合门户→办事大厅

5. **信息更新**：如遇政策调整，以学校最新发布的通知为准，学生应及时关注教务处官网和学院通知

---

## 九、特殊情况说明

1. **退役学生**：退役复学学生因特殊情况需请假，按正常流程办理，可适当放宽审批条件

2. **休学创业学生**：休学创业期间不适用本请假制度，按休学管理规定执行

3. **跨校区请假**：广州校区与佛山校区学生请假流程一致，跨校区参加活动需提前向学院报备

4. **毕业年级学生**：毕业年级学生请假应考虑毕业论文、实习等特殊安排，避免影响毕业进度
