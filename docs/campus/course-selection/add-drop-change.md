## 二、适用范围

广东财经大学普教本科生（含2022-2025级）正常学期课程的**退课、补选、改选**操作，包括主修、重修、辅修、跨专业/跨年级选课等场景的退补改规范。

## 三、退补改核心时间节点

| 轮次 | 退补改权限 | 核心规则 | 特殊说明 |

|------|------------|----------|----------|

| 第一轮 | **禁止退预置课**，仅可新增选课 | 不限人数、按规则筛选（抽签） | 预置课为行政班必修课，原则上不可退换{insert\\_element\\_0\\_} |

| 第二轮 | 开放退选、补选、改选 | 退选分开、先选先得，开放跨年级选课 | 结束后进行关班处理{insert\\_element\\_1\\_} |

| 第三轮 | 开放退选、补选、改选 | 退选分开、先选先得，开放跨专业选课 | 开放重修、辅修、转专业学生冲突选课{insert\\_element\\_2\\_} |

| 第四轮 | 开放退选、补选、改选（试听周） | 先选先得，与第三轮规则一致 | 下学期第1-2周进行，最后调整机会{insert\\_element\\_3\\_} |

| 选课结束后 | **禁止所有退补改操作** | 课表最终锁定，成绩录入生效 | 特殊情况需院系+教务处审批{insert\\_element\\_4\\_} |

## 四、退课规则详解

### 1. 退课权限分级

- **行政班必修课**：原则上**不可退课**，特殊情况（如疾病、转专业）需提交书面申请，经院系审批后报教务处备案{insert\\_element\\_5\\_}

- **教学班必修课**（思政二类四史、体育课、分层分类数学英语）：第二、三、四轮可自主退课{insert\\_element\\_6\\_}

- **选修课**（通识选修课、专业选修课）：第二、三、四轮可自主退课{insert\\_element\\_7\\_}

### 2. 退课关键限制

1. 退课必须在**对应轮次规定时间内**完成，逾期系统自动关闭退课功能{insert\\_element\\_8\\_}

2. 第四轮退课需先缴清前三轮全部课程学分学费，缴费后约1小时方可登录系统操作

3. 冲突选课申请通过后又退课的，系统**拒绝再次申请**该课程{insert\\_element\\_9\\_}

4. 试听期（第四轮）之后退选课程的，学校**全额收取**该课程学分学费

5. 退课后需确保学期总学分不低于培养方案最低要求

## 五、补选规则详解

### 1. 补选适用场景

1. 第一轮抽签未中，需补选其他教学班

2. 关班后被退课，需重新选择课程

3. 试听后发现课程不适合，需更换其他课程

4. 培养方案调整，需补选新增课程要求

### 2. 补选操作规范

1. 补选与退选同步进行，遵循**先选先得**原则，名额满即止{insert\\_element\\_10\\_}

2. 跨年级、跨专业补选需在对应轮次开放后进行{insert\\_element\\_11\\_}

3. 补选课程需注意**时间冲突**，非特殊情况（重修、辅修、转专业）不可冲突选课{insert\\_element\\_12\\_}

4. 补选成功后，系统实时更新课表，需及时核对确认{insert\\_element\\_13\\_}

## 六、改选规则详解

### 1. 改选本质

改选=退课+补选，需先退掉原课程，再补选新课程，遵循退课和补选的双重规则{insert\\_element\\_14\\_}

### 2. 改选特殊场景

1. **体育课改选**：仅可在对应选课轮次内进行，退课后重新选择项目{insert\\_element\\_15\\_}

2. **重修课程改选**：第三轮及以后开放，需选择"重修"类型{insert\\_element\\_16\\_}

3. **辅修课程改选**：第三轮及以后开放，需选择对应辅修专业课程{insert\\_element\\_17\\_}

4. **冲突改选**：仅限重修、辅修、转专业学生，需院系审批后操作{insert\\_element\\_18\\_}

## 七、退补改边界规则与注意事项

1.  **学分限制**：每学期主修课程原则上不超过34学分，辅修/转专业学生不超过38学分{insert\\_element\\_19\\_}

2.  **关班影响**：普通课程少于25人、专业选修课少于15人、体育课少于10人原则上关班，被关班学生需在后续轮次重新选课{insert\\_element\\_20\\_}

3.  **费用关联**：前三轮选课结束后需按财务处通知缴纳学费，第四轮退课不影响已缴费用

4.  **违规处理**：严禁使用脚本、外挂等违规手段进行退补改操作，一经发现严肃处理{insert\\_element\\_21\\_}

5.  **特殊审批**：选课结束后确需调整课程的，需提交书面申请，经任课教师、院系教学秘书、教学副院长签字后，报教务处审批{insert\\_element\\_22\\_}
