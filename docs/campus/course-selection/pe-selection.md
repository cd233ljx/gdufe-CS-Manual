## 二、适用范围

广东财经大学普教本科在校生，体育必修课程选课、项目选择、修读限制、保健课申请、考核与学分达标相关管理规则。

## 三、学分与修读基本要求

1. 体育为全校统一必修课，本科阶段需修满**总计8学分**，单门体育课程2学分。

2. 原则上安排在大一、大二阶段完成修读，分四个学期修读完毕。

3. 每学期仅限选修**1门**体育选项课，不允许多选、叠加选课。

4. 体育学分为毕业硬性要求，未修满规定学分，不予毕业。

## 四、选课轮次与班级规则

1. 体育选课纳入全校统一四轮选课体系，跟随各轮次时间同步开放。

2. 首轮多采用抽签模式分配热门体育项目，后续轮次为先选先得。

3. 体育教学班有人数容量限制，同时执行关班标准，选课人数不足10人原则上予以关班。

4. 教学班额满即止，无额外扩容权限，需自行选择剩余余量项目。

## 五、项目选择与重复修读限制

1. 开设球类、格斗、舞蹈、健身、水上等多类体育选修项目，供学生自主选择。

2. 同一体育项目、同名称课程考核合格后，**禁止重复修读**，不得重复刷取学分。

3. 课程考核不合格、缺考、违纪无补考资格的，可按重修流程重新选报对应体育课程。

4. 不得跨项目随意顶替修读，体育课程归属分类以教务处公布目录为准。

## 六、跨校区选课硬性限制

1. 严格实行**同校区修读**规则，禁止跨校区选报体育课程。

2. 跨校区体育选课一经核查，系统将做清课处理，选课结果无效，不计入学分。

3. 学生仅可选择本人就读校区开设的体育教学班。

## 七、保健课特殊申请规则

1. 适用对象：因身体疾病、伤病、特殊健康状况，无法参与常规剧烈体育运动的学生。

2. 申请要求：需提供正规医疗证明，按学校统一流程提交审批。

3. 修读规范：获批后仅限选修保健类体育课程，不得选报普通运动类项目；普通学生不得擅自选报保健课。

4. 保健课学分效力与常规体育课一致，正常计入体育必修总学分。

## 八、考核与违规边界规则

1. 体育课程综合成绩由平时考勤、课堂表现、体质测试、期末考核等部分综合评定。

2. 无故缺考、考场违纪、违规选课等行为，按全校考试管理规定处置，取消当次成绩及补考资格。

3. 体育课程退补改严格遵循全校选课统一时间安排，选课周期结束后，无特殊审批不得调整。

4. 时间冲突仅允许符合条件的重修、辅修、转专业学生申请冲突选课，普通学生体育课程不开放冲突审批。
