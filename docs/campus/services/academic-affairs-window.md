## 二、适用范围

广东财经大学全日制普通本科在校生（广州校区与佛山校区），适用于**在读证明/学籍证明、成绩单打印、学生证补办、请假/缓考、转专业、辅修申请、课程免听、毕业审核**等常见教务事项，不含研究生、继续教育学生及二级学院特殊规定。

## 三、核心教务事项办事流程

### （一）在读证明/学籍证明办理

1. **办理材料**：
- 本人身份证原件（代办需提供代办人身份证+本人授权说明）
- 学籍证明申请表（教务处官网下载）

2. **办理方式**：
- 线上：教务管理系统→"学籍管理"→"在读证明申请"→上传材料→等待审核→自行打印
- 线下：携带材料至所在学院教学秘书→审核后统一报送教务处→领取盖章证明

3. **办理周期**：线上即时生成，线下1-2个工作日

4. **收费标准**：免费

### （二）成绩单打印/申请

1. **办理材料**：
- 本人身份证原件
- 成绩单申请表（需注明打印份数和用途）

2. **办理方式**：
- 自助打印：广州校区明德楼305室、佛山校区厚德楼208室自助打印机，刷校园卡/身份证打印
- 人工办理：携带材料至教务处学籍科→审核→打印盖章

3. **办理周期**：自助打印即时完成，人工办理当日办结

4. **收费标准**：自助打印免费，人工打印**1元/份**

### （三）学生证补办（详见《学生证补办指南》）

1. **补办分类**：遗失补办、破损换发、信息变更补办、单独优惠卡补办

2. **核心流程**：填写申请表→学院统一汇总→教务处集中办理→学院领取下发

3. **补办周期**：材料收齐后1-2周

4. **收费标准**：工本费**3元/本**，优惠卡单独补办按公示标准收费

### （四）请假/缓考申请（详见《请假与缓考申请流程指南》）

1. **请假**：企业微信"学工通"线上申请为主，按请假时长分级审批

2. **缓考**：考前线上提交申请，突发情况考后2日内提交，随下一学期补考进行

3. **关键限制**：补缓考试原则上不得再申请缓考

### （五）转专业申请

1. **申请条件**：大一上学期学业成绩专业排名前30%，无违纪处分

2. **申请时间**：每年3月和9月集中受理

3. **办理流程**：
- 填写《转专业申请审批表》→学院初审→教务处审核→转入学院考核→公示→学籍异动

4. **注意事项**：艺术类、体育类专业不得转入普通类专业，中外合作办学专业仅限同项目内转专业

### （六）辅修专业申请

1. **申请条件**：主修专业成绩良好，无不及格课程

2. **申请时间**：大二上学期9-10月

3. **办理流程**：
- 登录教务系统→"辅修申请"→选择专业→学院审核→缴费注册→开始修读

4. **收费标准**：按学分收费，每学分**100元**

### （七）课程免听申请

1. **申请条件**：已修读过相同或相近课程且成绩≥80分，或有相关职业资格证书

2. **办理流程**：
- 填写《课程免听申请表》→任课教师签署意见→学院审核→教务处备案

3. **注意事项**：思政课、体育课、实验课、实践课不得申请免听

### （八）毕业及学位授予审核

1. **申请时间**：毕业前一学期10-11月

2. **办理流程**：
- 学生提交毕业/学位申请→学院初审→教务处审核→学位评定委员会审批→证书发放

3. **关键规则**：毕业时不授予学位的，毕业后一律不补授学位

## 五、通用办事规则与注意事项

1. **集中办理原则**：学生证补办、转专业、辅修等事项实行集中统一办理，不接受个人零散递交

2. **材料规范**：所有申请材料需真实有效，复印件需注明"与原件一致"并签名

3. **线上优先**：请假、缓考、在读证明等事项优先通过企业微信或教务系统线上办理

4. **时限要求**：各类申请需在规定时间内提交，逾期一般不予受理

5. **代办规定**：原则上需本人办理，特殊情况可代办，须提供代办人身份证和本人授权说明

## 六、常见问题FAQ

### （一）通用问题

Q1：所有教务事项都必须本人办理吗？

A1：原则上需本人办理，特殊情况可代办，须提供代办人身份证和本人授权说明

Q2：线上申请和线下申请有什么区别？

A2：线上申请更快捷，可随时提交和查询进度；线下申请适用于复杂或特殊情况，需现场审核材料

### （二）学籍与成绩

Q3：成绩单自助打印和人工打印有什么不同？

A3：自助打印无教务处公章，适用于个人参考；人工打印有公章，适用于升学、就业等正式用途

Q4：转专业后原专业的课程成绩还有效吗？

A4：已修课程中与转入专业教学计划相同或相近的课程，成绩合格者可按规定转换学分

### （三）考试与教学

Q5：课程免听后还需要参加考试吗？

A5：需要，免听仅免除听课环节，必须参加课程考核并达到合格标准才能获得学分

Q6：毕业审核未通过怎么办？

A6：可申请结业，在规定年限内完成学业后换发毕业证书；学位未通过者，毕业后不再补授
