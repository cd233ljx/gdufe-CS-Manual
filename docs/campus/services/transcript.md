## 三、自助打印办理流程（推荐，24小时服务）

### （一）适用范围

- 全日制普通本科生、研究生（含佛山校区）

- 毕业未满1年的本科/研究生毕业生

- 仅支持**中文成绩单、成绩证明**打印

### （二）操作步骤

1. **前往自助终端**：各校区自助打印机位置如下（24小时开放）
- 广州校区：教师办公楼一楼大厅、二号楼一楼大厅、校友楼一楼大厅
- 佛山校区：学生活动中心一楼电梯口、厚德楼一楼大厅

2. **身份验证**（三选一）：
- 校园卡刷卡
- 身份证刷卡
- 融合门户账号密码登录（研究生初始密码为身份证后六位）

3. **选择打印类型**：在主界面选择"成绩单"或"成绩证明"

4. **确认信息**：核对姓名、学号、专业、课程成绩等关键信息

5. **打印并领取**：系统自动生成带电子公章的正式成绩单，**即打即用**

6. **退出系统**：打印完成后务必退出账号，保护个人信息

### （三）注意事项

1. 自助打印的成绩单**与人工盖章版本具有同等效力**，可直接用于各类官方场合

2. 每人每天限打印**5份**，避免资源浪费

3. 打印前确保设备有纸，无纸时请联系网络中心（020-84096230）

4. 研究生打印成绩单需满足：**个人培养计划已通过学院审核且计划内无未修读、未通过或未完成课程**

## 四、线下人工办理流程（特殊情况）

### （一）适用场景

1. 需办理**英文成绩单**

2. 需办理**均分证明、排名证明**等个性化证明

3. 毕业超过1年的毕业生

4. 自助打印系统故障或无法满足特殊格式要求

5. 需办理**信封骑缝章**（用于留学申请）

### （二）办理步骤

1. **准备材料**：
- 本人有效身份证件（身份证原件，学生证辅助）
- 成绩单用途说明（如留学需提供目标院校模板）
- 英文成绩单需提供**中文原件+翻译件**（翻译件需与原件内容一致，外事处网站有模板）
- 均分/排名证明需填写《广东财经大学学生成绩证明申请表》（教务处网站下载）
- 委托代办需额外提供：学生手写授权书+代办人身份证原件

2. **提交申请**（按身份分类）：
- 在校本科生：教务处学籍科（广州校区教师办公楼2220室）
- 在校研究生：研究生院培养科（佛山校区厚德楼501室）
- 毕业未满1年：原就读学院教务办→教务处学籍科
- 毕业超过1年：档案馆（广州校区行政楼307室）
- 国际学院学生：国际学院教务办（善学楼507室）

3. **审核盖章**：
- 材料审核通过后，工作人员在规定时间内完成盖章
- 审核时间：周一至周五（不含周四下午），周四下午为集中领取时间

4. **领取证明**：
- 领取时间：周四下午14:30-16:15（提交材料后下一个周四）
- 领取地点：原提交材料的科室窗口

## 六、常见问题FAQ

### （一）自助打印相关

Q1：自助打印的成绩单有法律效力吗？

A1：有，自助打印的成绩单**带有电子公章**，与人工盖章版本具有同等法律效力，可直接用于各类官方场合

Q2：忘记融合门户密码无法登录自助终端怎么办？

A2：携带本人身份证到网络信息中心（广州校区行政楼307室）重置密码，或通过融合门户"忘记密码"功能找回

Q3：自助打印显示"无权限"是什么原因？

A3：可能原因：① 有未修读/未通过课程；② 培养计划未通过学院审核（研究生）；③ 已毕业超过1年；④ 系统故障。请联系教务处或研究生院解决

### （二）线下办理相关

Q4：英文成绩单如何办理？

A4：先通过自助终端打印中文成绩单，再按中文原件内容翻译（外事处网站有模板），将中文原件+翻译件一并提交至教务处学籍科审核盖章，周四下午领取

Q5：办理成绩单需要缴费吗？

A5：**免费办理**，自助打印和人工办理均不收取任何费用

Q6：毕业超过1年如何办理成绩单？

A6：携带本人身份证到档案馆（广州校区行政楼307室）办理，需提供毕业证/学位证原件备查

### （三）特殊情况处理

Q7：需要办理均分证明或排名证明怎么办？

A7：① 下载并填写《广东财经大学学生成绩证明申请表》；② 学院审核签字；③ 提交至教务处学籍科；④ 周四下午领取

Q8：委托他人代办成绩单需要什么材料？

A8：需提供学生本人手写授权书、学生身份证复印件、代办人身份证原件及复印件

Q9：成绩单上的信息有误怎么办？

A9：立即联系教务处学籍科（020-84096837）或研究生院培养科（0757-87200316）核实修改，修改后重新打印或办理

### （四）成绩单有效期与用途

Q10：成绩单的有效期是多久？

A10：一般为**3个月**，建议按需办理，不要提前太久打印

Q11：成绩单可以用于哪些场合？

A11：可用于国内升学、考研复试、公务员考试、入职背调、出国留学、交流项目、奖学金申请、学分认定等各类官方场合
