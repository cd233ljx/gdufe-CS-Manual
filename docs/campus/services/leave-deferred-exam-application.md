## 二、适用范围

广东财经大学全日制普通本科在校生，包含广州校区与佛山校区，适用于**日常请假、考试缓考**等相关申请场景，不含研究生、继续教育学生及二级学院特殊规定。

## 三、请假申请流程（企业微信线上为主）

### （一）请假类型与材料要求

| 请假类型 | 适用场景 | 必备材料 | 特殊说明 |

|----------|----------|----------|----------|

| **病假** | 身体不适需休息或治疗 | 校医院/三甲医院诊断证明、病历本 | 急诊可先口头报备，返校后48小时内补材料 |

| **事假** | 个人重要事务需处理 | 合理事由说明、家长知情同意书（如需） | 一般从严审批，旅游/娱乐等非必要事由不予批准 |

| **公假** | 学校组织的比赛/会议/实习等 | 官方活动通知、带队老师证明 | 可由辅导员/带队老师发起申请 |

| **离校假** | 周末/节假日离校外出 | 行程说明、紧急联系人信息 | 按时长分级审批 |

### （二）审批权限分级（2025年最新执行标准）

1. **2天内**：辅导员/班主任审批

2. **3-7天**：学院领导审批

3. **7天以上**：学院领导签署意见后，报送教务处审批

4. **一学期累计超6周**：需教务处审核并报学校领导备案{insert\\_element\\_0\\_}

### （三）企业微信"学工通"线上办理流程

1. **绑定准备**：确保已绑定广东财经大学企业微信（未绑定者先按官方链接完成绑定）

2. **发起申请**：
- 微信→通讯录→我的企业"广东财经大学"→"C5.学工通"→"学生管理"→"学工服务"→"学生请假"
- 填写请假类型、事由、起止时间、紧急联系人等必填信息
- 上传相关证明材料（病假上传诊断证明，事假上传相关说明）
- 点击"提交"，等待审批

3. **审批进度查询**："申请记录"→查看审批状态→"审批历史"→"查看审批流程"

4. **销假与续假**：
- 返校后及时在系统中完成销假操作
- 需续假的，点击原申请记录"延假"按钮，填写延假信息并提交审批

### （四）线下紧急办理流程（特殊情况适用）

1. 填写《广东财经大学普教本科生请假审批表》（教务处官网下载）

2. 按审批权限找辅导员、学院领导签字盖章

3. 提交至学院教学秘书处备案

4. 返校后及时补办线上流程并销假

## 四、缓考申请流程（线上线下并行）

### （一）申请条件与时间要求

1. **适用条件**：因特殊原因（疾病、突发事件等）无法参加期末考试的学生{insert\\_element\\_1\\_}

2. **申请时限**：
- 常规情况：**考前**提交申请{insert\\_element\\_2\\_}
- 突发情况：考后**2日内**提交，逾期不再受理{insert\\_element\\_3\\_}

3. **禁止情形**：补缓考试原则上不得再申请缓考{insert\\_element\\_4\\_}

### （二）申请材料（缺一不可）

1. 《广东财经大学普教本科课程缓考申请表》（教务处官网下载）{insert\\_element\\_5\\_}

2. 无法参加考试的证明材料（病假需医院诊断证明，事假需相关证明）{insert\\_element\\_6\\_}

3. 任课教师签署的意见（线下办理适用）{insert\\_element\\_7\\_}

### （三）线上办理流程（推荐）

1. 登录教务管理系统（http://jwxt.gdufe.edu.cn/jsxsd）

2. 进入"我的学业"→"缓考申请"→选择对应课程

3. 填写申请理由，上传缓考申请表和证明材料{insert\\_element\\_8\\_}

4. 等待任课教师、学院教学秘书、学院领导逐级审核

5. **成功标识**：课程成绩查询显示为"缓考"（显示"缺考"则为申请失败）{insert\\_element\\_9\\_}

### （四）线下办理流程（特殊情况适用）

1. 填写申请表并准备证明材料{insert\\_element\\_10\\_}

2. 找任课教师签署意见{insert\\_element\\_11\\_}

3. 提交至学院教学秘书处，由学院领导签署意见{insert\\_element\\_12\\_}

4. 教学秘书统一报送至教务处备案{insert\\_element\\_13\\_}

### （五）后续安排

1. 缓考成功后，随下一学期补考进行考试{insert\\_element\\_14\\_}

2. 考前登录教务系统查看打印个人考试安排表{insert\\_element\\_15\\_}

3. 缓考成绩按正常考试成绩记载，不额外扣分

## 六、常见问题FAQ

### （一）请假相关

Q1：可以只口头向辅导员请假吗？

A1：不可以，必须通过企业微信线上系统或填写纸质审批表完成正式请假手续，否则按旷课处理

Q2：请假后忘记销假会有什么后果？

A2：未按规定时间销假且未办理续假手续者，超出部分按旷课处理

### （二）缓考相关

Q3：缓考申请成功后，什么时候参加考试？

A3：随下一学期开学初组织的补考进行考试{insert\\_element\\_16\\_}

Q4：缓考成绩和正常考试成绩有区别吗？

A4：没有区别，缓考成绩按正常考试成绩记载，不额外扣分

Q5：补缓考试可以再申请缓考吗？

A5：不可以，补缓考试原则上不得再申请缓考{insert\\_element\\_17\\_}
