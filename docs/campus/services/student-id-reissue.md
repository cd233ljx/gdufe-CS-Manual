# 学生证补办指南

2. 本科生学籍异动及证件补办工作提示 | 2024-11-08 | 广东财经大学教务处

## 二、适用范围

广东财经大学全日制普通本科在校生，学生证遗失、破损、损坏、信息变更的补办与换发办理规则，不含研究生、二级学院自建规定。

## 三、补办分类

1. 遗失补办：学生证丢失、损毁无法正常使用

2. 破损换发：外壳破损、页面脱落、打印模糊，需上交旧证

3. 信息变更补办：转专业、学籍调整等导致证件信息不符

4. 单独优惠卡补办：仅火车票优惠卡失效，学生证本体完好

## 四、补办材料

1. 学生证补办申请表（教务处官网下载最新版）

2. 本人一寸证件照2张

3. 本人身份证原件

4. 破损换发须交回原损坏学生证

5. 代办需提供代办人身份证+本人授权说明

## 五、办理流程

1. 个人填写申请表，准备齐全材料

2. 提交至所在学院教学秘书统一汇总审核

3. 学校实行**集中统一办理**，不接受学生个人零散单独递交

4. 教务处定期统一印制、盖章，完成后由学院统一领取下发

5. 补办周期：材料统一收齐后，1–2周内办结

## 六、收费标准

1. 学生证补办工本费：**3元/本**

2. 火车票优惠卡单独补办：按学校统一公示标准收费

3. 所有证件补办仅限校内官方渠道，无第三方收费

## 七、关键规则限制

1. 无临时加急补办通道，严格按集中批次办理

2. 学生证为校内法定身份凭证，严禁私自涂改、自制、伪造

3. 证件遗失须及时报备学院，避免冒用

4. 优惠卡每年仅限规定时段充值、补办，非办理期不予受理

5. 已毕业学生不再补办学生证

## 八、常见FAQ

Q1：可以自己直接去教务处现场补办吗？

A1：不可以，须由学院统一收集材料上报，个人不单独受理。

Q2：旧学生证丢了，还要上交吗？

A2：遗失补办无需上交；破损换发必须回收旧证。

Q3：转专业后信息不对，需要重新补办吗？

A3：需要，凭学籍异动审批材料，按信息变更流程统一补办。

Q4：优惠卡消磁怎么处理？

A4：不用重办整本学生证，走单独优惠卡补办流程即可。
