## 二、适用范围

广东财经大学全体全日制普通本科在校生（广州校区与佛山校区统一执行），**不含研究生、继续教育学生**。适用于所有法定节假日及寒暑假期间的教学安排调整，包括：

1. 元旦、春节、清明节、劳动节、端午节、中秋节、国庆节等法定节假日

2. 寒暑假期间的开学、放假及调课安排

3. 因特殊情况（如新生军训）导致的教学时间调整

## 四、通用放假调课规则（近三年统一）

### （一）放假与调课基本原则

1. 严格按照**国务院办公厅通知**和学校实际情况安排节假日放假与调课

2. 法定节假日调休原则：**周六/周日补工作日课程**，确保教学计划顺利完成

3. 寒暑假安排：每学期教学周数固定为**19周**，寒暑假时间根据校历统一安排

4. 新生军训调整：2024级、2025级新生因军训安排在第17-18周，考试和放假时间做相应调整

### （二）教学安排调整要求

1. 调课期间，所有课程**严格按照补课安排表**进行，不得擅自更改

2. 任课教师应提前通知学生补课时间和地点，确保学生按时参加

3. 因调课导致的课程冲突，学生应及时联系所在学院教学秘书协调解决

4. 节假日期间，图书馆、实验室等教学资源开放时间按学校通知执行

### （三）假期安全与管理

1. 学生假期离开学校前，应**妥善保管个人财物**，关闭宿舍水电，锁好门窗

2. 假期出行应遵守交通规则，注意人身安全，避免前往危险地区

3. 学生应保持通讯畅通，遇紧急情况及时与学校和家人联系

4. 假期结束后，学生应按时返校报到注册，**逾期未报到按学校规定处理**

### （四）信息查询方式

1. 放假调课通知由学校办公室统一发布在**广东财经大学官网**首页通知公告栏

2. 学生可通过学校官方微信公众号、教务处网站查询最新放假调课信息

3. 各学院教学秘书会通过班级群等渠道通知学生具体调课安排

4. 如有疑问，可联系所在学院教学秘书或学校办公室咨询

## 五、重要注意事项

1. **时间刚性**：放假和调课时间严格按通知执行，**不得擅自更改**，违者按学校规定处理

2. **考勤要求**：调课期间的课程考勤与正常上课考勤标准一致，**缺席按旷课处理**

3. **考试安排**：法定节假日期间**不安排任何考试**，如有特殊情况另行通知

4. **补考影响**：放假调课不会影响正常补考安排，补考时间以教务处通知为准

5. **咨询方式**：
- 学校办公室咨询电话：020-84096888（广州校区），0757-87200000（佛山校区）
- 教务处咨询电话：020-84096837（广州校区），0757-87200316（佛山校区）
- 工作时间：周一至周五 9:00-12:00，14:30-17:00

6. **信息更新**：如遇特殊情况（如自然灾害、疫情等），学校将及时调整放假调课安排并发布通知

## 六、特殊情况说明

1. **新生军训调整**：2024级、2025级新生因军训安排在第17-18周，学校集中考试时间做了相应调整，具体以教务处通知为准

2. **跨校区协调**：广州校区与佛山校区放假调课安排**完全统一**，学生不得跨校区参加补课

3. **实践课程调整**：实验、实习、体育等实践性课程的放假调课安排由各开课学院自行确定并通知学生

4. **毕业生安排**：应届毕业生的放假调课安排可能因毕业实习、毕业论文等原因有所调整，以学院通知为准
