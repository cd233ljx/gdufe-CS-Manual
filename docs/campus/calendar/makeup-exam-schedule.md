## 二、适用范围

广东财经大学全日制普通本科在校生（广州校区与佛山校区统一执行），**不含研究生、继续教育学生**。适用于：

3. 其他符合学校规定可参加补（缓）考的情况

## 四、通用补（缓）考规则（近三年统一）

### （一）补考资格

1. 上一学期期末考试**成绩不及格**且未申请缓考的学生（特殊课程除外）

2. 已按规定申请**缓考并获批准**的学生（缓考申请成功标识：课程成绩查询显示为"缓考"）

3. **以下课程不安排补考**：实验、实习、体育、演奏等实践性课程，此类课程不及格需重修

4. 未按规定办理缓考手续缺考的学生，按**旷考**处理，**不得参加补考**

### （二）缓考申请流程

1. **线下申请**：
- 下载《广东财经大学普教本科课程缓考申请表》（教务处官网下载）
- 填写并附上相关证明材料（如医院诊断证明、事假证明等）
- 任课教师签署意见后交至学生所在学院教学秘书处
- 学院领导签署意见后完成申请

2. **线上申请**：
- 部分学期支持通过教务综合信息集成服务平台申请
- 申请时间一般为考前一周内，逾期未提交视为自动放弃
- 审批结果将在规定时间内公布，审批通过后方可参加缓考

### （三）考试安排查询

1. 补（缓）考具体安排分为**集中考试**和**分散考试**两类

2. 集中考试安排表由教务处统一发布在官网通知附件中

3. 分散考试安排由各开课学院自行确定并通知学生

4. 学生可通过教务管理系统查询个人补（缓）考安排

### （四）考试要求

1. 考生必须携带**学生证/身份证**等有效证件参加考试

2. 严格遵守考场纪律，违纪者按《广东财经大学学生违纪处分规定》处理

3. 补（缓）考成绩按实际得分记录，**标注"补考"或"缓考"字样**

4. 补考仍不及格的课程，需按规定**重修**

---

## 五、重要注意事项

1. **时间刚性**：补（缓）考时间严格按通知执行，**逾期未参加视为自动放弃**，不再另行安排

2. **信息核对**：学生应在规定时间内核对个人补（缓）考资格及安排，如有疑问及时联系学院教学秘书

3. **申请时限**：缓考申请必须在**考前**完成，考后申请不予受理

4. **重修安排**：补考不及格或未参加补考的学生，需在后续学期按学校规定申请重修

5. **咨询方式**：
- 教务处咨询电话：广州校区020-84096837，佛山校区0757-87200316
- 工作时间：周一至周五 9:00-12:00，14:30-17:00

6. **信息查询**：最新补（缓）考通知及附件可通过广东财经大学教务处官网（https://jwc.gdufe.edu.cn/4133/list.htm）查询

## 六、补（缓）考成绩管理

1. 补（缓）考结束后，任课教师应在**一周内**完成成绩录入

2. 成绩录入后，学生可通过教务管理系统查询补（缓）考成绩

3. 应届毕业生补（缓）考成绩需优先录入，确保不影响毕业资格审核

4. 补（缓）考成绩与正常考试成绩具有同等效力，计入学生学业档案
