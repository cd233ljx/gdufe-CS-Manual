## 二、适用范围

广东财经大学普教本科生正常期末考核、补考、缓考全过程中，**无故缺考、违规违纪、考试作弊**等行为的规则约束与后果认定。

## 三、无故缺考相关规则

1. 未按规定办理缓考手续，无故不参加课程正常考试、补考、缓考，一律认定为**缺考**。

2. 缺考课程当次成绩记为0分，**取消该课程补考资格**。

3. 补（缓）考无故缺席，同样按缺考标准处理，不再额外安排补测。

4. 缺考课程无法通过补考补救，只能选择后续重修修读。

5. 分散类考核未在规定时限内提交答卷，逾期统一按缺考处理。

## 四、考试违纪行为后果

1. 考场违规、违反考试纪律等一般违纪行为，当场取消本次考试成绩。

2. 携带手机、智能设备、违禁资料进入考场等违规情形，按违纪从严认定。

3. 违纪记录计入学业档案，作为学业预警、学籍管理的参考依据。

4. 违纪课程取消补考资格，仅可通过重修完成学分修读。

## 五、考试作弊严重后果

1. 考试作弊一经查实，当次考核成绩无效，直接记0分。

2. 取消该门课程补考、缓考资格，必须重修。

3. 依据学校管理制度给予相应纪律处分，处分材料存入个人档案。

4. 严重作弊行为会影响评优评先、奖助申请、学位授予等资格。

## 六、通用边界规则

1. 所有缺考、违纪、作弊处置标准，统一以教务处官方考试通知为准。

2. 已获批缓考的学生，必须按时参加统一补缓考，缺席仍按缺考处理。

3. 各类考试全程实行证件核验，无证件、违规带入物品均视为考场违规。

4. 成绩、考核资格限制一经录入教务系统，原则上不予更改。
