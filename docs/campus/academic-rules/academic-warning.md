## 二、适用范围

广东财经大学普教本科学生**学业预警、学籍预警**的触发、分级、处置与跟踪管理工作。

## 三、预警触发条件

1. **成绩类预警**

    平均学分绩点＜2.0；多门课程考核不合格、未修满规定学分。

2. **考勤类预警**

    单学期无故旷课≥10节；长期未参与课程学习、考勤不达标。

3. **学籍类预警**

    逾期未完成报到注册；即将超过学校规定的最长学习年限。

4. **处分/违规类预警**

    因考试违纪、旷考、取消考试资格等导致成绩异常，影响学业推进。

## 四、预警等级分类

实行**三级预警机制**（蓝、黄、红），按学业风险程度分级处置：

1. **蓝色预警**：轻度学业异常，提醒整改。

2. **黄色预警**：中度学业风险，督促帮扶。

3. **红色预警**：重度学业危机，启动专项处置。

## 五、预警处置流程

1. 学校定期统计学生成绩、考勤、学籍状态，自动/人工触发预警。

2. 学院向学生及家长送达预警通知，明确问题与整改要求。

3. 学生按要求完成课程重修、补修、考勤整改等学业补救。

4. 学院跟踪整改效果，预警解除或升级处置。

5. 超最长学习年限、逾期未整改的，按学籍管理规定做退学等处理。

## 六、重要边界规则

1. 学业预警结果作为**学籍处理、毕业审核**的重要依据。

2. 预警期间需按时完成学业整改，否则预警等级升级。

3. 逾期注册、未报到将直接触发学籍预警，逾期不处理影响学籍保留。

4. 预警信息录入学生学业档案，全程留痕可查。
