# GDUFE CS Manual · 广东财经大学实用指南

一个由广东财经大学计算机专业学生共建的实用指南网站，提供课程资源、校园生活、工具箱等全方位信息。

## 🌟 项目简介

GDUFE CS Manual 是一个专为广东财经大学学生打造的综合指南平台，旨在帮助学生更好地适应大学生活、掌握专业知识、规划未来发展。

- **课程资源**：涵盖基础课程、专业核心课和选修推荐，提供详细的学习资料和经验分享
- **校园生活**：包含美食地图、通勤指南、宿舍生活等实用信息
- **工具箱**：提供各种学习和生活工具推荐
- **FAQ**：解答常见问题
- **贡献指南**：鼓励学生参与内容共建

## 📚 主要功能

- **课程资源**：详细的课程资料和学习经验分享
- **校园生活**：广州校区和三水校区的生活指南
- **美食地图**：校内校外美食推荐
- **通勤指南**：校区间交通信息
- **工具箱**：实用工具推荐
- **FAQ**：常见问题解答

```
├── docs/             # 文档内容

│   ├── courses/      # 课程资源

```
